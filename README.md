# HMTT

Household Monetary Track & Treaure

# Dependencies

## pip

Check [pip-installation](https://pip.pypa.io/en/stable/installation/) for more detail

```bash
python -m ensurepip --upgrade
```

## Poetry

```bash
pip install poetry
```

# Install Application Dependencies

```bash
poetry install
```

# Run

```bash
poetry run ./bin/htmm
```
